# Сайт — Резюме та Аналітика (готово для GitHub Pages)

Це статичний сайт з трьома сторінками українською мовою:
- `index.html` — Резюме (з інформацією з резюме користувача).  
- `security.html` — Короткий аналіз економічної безпеки США (надані дані).  
- `threats.html` — Загрози та методи протидії.

Інструкції:
1. Розпакуйте архів та завантажте в репозиторій GitHub (наприклад, у гілку `main`).
2. У проекті GitHub зайдіть у **Settings → Pages** і виберіть гілку `main` та папку `/ (root)`.
3. Після збереження сайт буде доступний через GitHub Pages.

Файли у пакеті:
- index.html
- security.html
- threats.html
- styles.css
- script.js
- README.md

Контактна інформація у файлі `index.html` узята з резюме: anasttn@icloud.com.
